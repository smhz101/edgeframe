A functional-first WordPress parent theme framework with clean hooks and minimal classes.

== Description ==
EdgeFrame is a developer-friendly base theme providing:
- Schema-driven admin settings (tabs, sections, fields)
- Clean template structure
- Accessible markup and sensible defaults

== Installation ==
1. Upload the `edgeframe` theme to your `/wp-content/themes/` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Visit Appearance â†’ EdgeFrame Settings to configure options

== Frequently Asked Questions ==
= Where are shortcodes and security tweaks? =
Shortcodes and security header tweaks are plugin territory per Theme Directory guidelines. Use a small site plugin to add them.
